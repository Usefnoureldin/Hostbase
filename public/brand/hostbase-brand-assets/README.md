# Hostbase Brand Assets

Welcome to the official Hostbase Brand Asset Kit.

**What's inside:**
- `/logos/full-logo/`: 5 files containing the logomark and wordmark combined side by side.
- `/logos/logomark/`: 3 files containing only the connected-node 'H' logomark.
- `/logos/wordmark/`: 3 files containing only the "Hostbase" text.
- `/logos/app-icon/`: 4 files optimized for 512x512 app rendering and 32x32 favicons.
- `/guidelines/hostbase-brand-guidelines.md`: The complete rulebook for Hostbase color, typography, messaging, and acceptable logo use.

All assets are perfectly vectorized SVGs ready for production web/print usage.
