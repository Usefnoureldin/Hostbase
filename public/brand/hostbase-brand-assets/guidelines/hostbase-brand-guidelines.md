## Typography

Primary typeface: **Inter**

- **H1**: 48px / 600 weight / 1.2 line height / -0.02em letter spacing
- **H2**: 36px / 600 weight / 1.3 line height / -0.01em letter spacing
- **H3**: 24px / 500 weight / 1.4 line height 
- **H4**: 20px / 500 weight / 1.4 line height
- **Body**: 16px / 400 weight / 1.6 line height
- **Caption**: 12px / 400 weight / 1.5 line height 
- **Button**: 14px / 500 weight / 1.0 line height / 0.5px letter spacing

## Logo Usage Rules

**Minimum clear space:** 
Always preserve a clear space around the logo equal to 1x the height of the logomark 'H'.

**Minimum sizes:**
- Full logo: 120px wide
- Favicon/Logomark alone: 24px wide

**Do NOT:**
- Do not stretch, rotate, or apply perspective.
- Do not add drop shadows, outer glows, or other effects.
- Do not change or edit colors. Use only the provided palettes.
- Do not rearrange the logomark and wordmark. They must always sit side by side as provided (unless using app/favicon).

## Brand Voice

- **Tone**: Calm, confident, understated. Quiet authority — never salesy or hype-driven.
- **Personality**: Sophisticated, precise, operator-first. Built by someone who runs properties, not just builds software.
- **We say**: "your brand," "under your control," "all-in-one," "automated," "white-label"
- **We don't say**: "disrupting," "revolutionizing," "game-changing," "AI-powered" as a headline (the AI is in the product, not the marketing)
- **Writing style**: Short sentences. Direct. No jargon unless the audience uses it daily. Show, don't tell.
